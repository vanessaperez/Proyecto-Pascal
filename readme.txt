                             MAZE 
                            ------


    El objetivo de este juego es completar el laberinto,
llevando al jugador (P) hasta la zona de salida (E) al final 
del camino sin tocar las paredes (#) y sin ser alcanzado por 
los monstruos (@) que se encuentran en el laberinto.

    Las dificultades del juego son: fácil, difícil, extremo 
y un último nivel el cual es el más complicado de todos.  


 			INSTRUCCIONES
------------------------------------------------------------

   1.- Compile el archivo con la instrucción fpc maze.pas

   2.- Ejecute el archivo con el comando ./maze

   3.- Escoja la dificultad seleccionando la letra correcta.

   4.- Para moverse dentro del laberinto utilice las teclas:
       (deben estar en minúscula)

	j: para moverse a la izquierda.
        l: Para moverse a la derecha.
	k: para moverse hacia abajo.
 	i: para moverse hacia arriba.

NOTA: los archivos .txt de los laberintos deben estar en la 
misma carpeta que el ejecutable ./maze
